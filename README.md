# Restart Raycast

This Raycast extension allows you to restart Raycast, releasing its RAM usage.

## Usage

1. Clone the repository locally on your machine.

2. Open Raycast and go to "Import Extension".

3. Run `npm install` and `npm run build` to activate the extension.

4. Launch the "Restart Raycast" command directly from Raycast.

### Notes

- This extension works **on macOS only**.
